Important Git Commands

Configure Git

  • git config --global user.name "Your_Name"
  • git config --global user.email "Your_email"
  • git config --list

Cloning a repository

To clone an existing git repository from another computer or online platform:

  • git commit "https://(link)"

Git Status

It displays the state of code

  • git status

Git Add

It adds new or changed files in your working directory to git staging area.

  • git add file_name
To add all files :
  • git add .

Git Commit

It is the record of change

  • git commit -m "some message"

Git Push

To upload local repository content to remote repository

  • git push origin main

Git Init

Powers your folder to be managed by git, and initialises a new empty repository

  • git init
  • git remote add origin ---link---
  • git remote -v
  • git branch
  • git branch -M main
  • git push origin main

Git Add & Commit

To add and commit at same time

  • git commit -am "message"

Git Upstream

To set up stream

  • git push -u origin main

Branch Commands

To check branch

  • git branch

To remove branch

  • git branch -M main

To navigate among branches

  • git checkout ---branch_name---

To create new branch

  • git checkout -b ---new_branch_name---

To delete branch

  • git branch -d ---branch_name---

Merging Branches

To compare commits, branches,files & more

  • git diff ---branch_name---

To merge two branch

  • git merge ---branch_name---

Pull Command

To pull changes from remote repository to local repository

  • git pull origin main

Git Log

Shows all commits

  • git log

Fixing Mistakes

case-1 : staged changes

Added unnecessary changes (to undo changes)

  • git reset ---file_name---
  • git reset

case-2

Commited changes (for one commit undo)

  • git reset HEAD~1

case-3

Commited changes (for many commit undo)

  • git reset ---commit_hash---
  • git reset --hard ---commit_hash---
